# Projeto TypeScript Básico

Um projeto TypeScript básico para começar a desenvolver.

## Pré-requisitos

- Node.js (versão 16 ou superior)
- npm ou yarn

## Instalação

```bash
npm install
```

## Scripts Disponíveis

- `npm run build` - Compila o projeto TypeScript para JavaScript
- `npm start` - Executa o arquivo compilado
- `npm run dev` - Executa o projeto em modo desenvolvimento (sem compilar)
- `npm run watch` - Compila e observa mudanças nos arquivos

## Estrutura do Projeto

```
.
├── src/           # Código fonte TypeScript
│   └── index.ts   # Arquivo principal
├── dist/          # Código compilado (gerado automaticamente)
├── tsconfig.json  # Configuração do TypeScript
└── package.json   # Dependências do projeto
```

## Como Usar

1. Instale as dependências:
```bash
npm install
```

2. Execute o projeto:
```bash
npm run dev
```

Ou compile e execute:
```bash
npm run build
npm start
```
